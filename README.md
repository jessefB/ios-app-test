# Overview

This app is a proof of concept for a text-to-speech generator. Enter the text you want to be spoken, adjust speed and select the language, and press 'Say it'.

[Software Demo Video](https://youtu.be/2tvR40wcm-M)

# Development Environment

This app was built in XCode on an Apple Mac running Big Sur. The `AVFoundation` library was used for the text-to-speech generation.
